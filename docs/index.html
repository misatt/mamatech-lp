本当にITが苦手でも大丈夫ですか？

大丈夫です！プログラミングのような難しいことは一切やりません。普段使っているスマホやパソコンができれば十分です。AIを活用して、難しい作業は自動化していきます。

学習時間はどのくらい必要ですか？

1日30分〜1時間程度を目安にしています。子どもが寝た後の「夜活」時間を活用して、無理のないペースで進められます。

実際にどのくらい稼げるようになりますか？

個人差はありますが、卒業生は月1万円〜5万円程度の収入を得ています。「爆発的な収入」ではなく、「ゆるく・長く」続けられる範囲での収入を目指します。

子どもが小さくても参加できますか？

むしろ小さなお子様がいるママにこそ参加していただきたいプログラムです。同じ境遇の仲間と支え合いながら、無理のないペースで進められます。
